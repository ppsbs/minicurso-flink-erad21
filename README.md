# Flink Tutorial

## Requirements
- Java 11
- Maven > 3.6
- IntelliJ or Eclipse


## Code
Clone the repository:
   ```bash
   git clone git@github.com:ppsbs/minicurso-flink-erad21.git
   ```

If you are using eclipse, you might have to run the following command to open the project correctly:
```bash
mvn eclipse:eclipse
```
